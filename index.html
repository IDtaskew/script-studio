Paste your Cloudflare Worker URL. Saved locally — not shared. Setup guide ↓
One-time setup — takes about 5 minutes 1. Go to dash.cloudflare.com/workers → Create Worker → paste the code from worker/index.js in this repo → Deploy.
2. In your Worker settings → Variables → add a secret named ANTHROPIC_API_KEY with your key from console.anthropic.com.
3. Copy your Worker URL (e.g. https://script-studio-api.yourname.workers.dev) and paste it into the field above.
4. Click Save. You're done. Your API key never touches this page.
